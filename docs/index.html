AO Design Tokens

A collection of tokens for the AO Design System. These tokens support light and dark mode and are based on the Radix Color System.

  1. To get started, install the package from npm: 
                  npm install ao-design-tokens
  2. Import the tokens in your project: 
                  @import "ao-design-tokens";
  3. Use the tokens in your CSS: 
                  .my-element { background-color: var(--colors-pink-9); color: var(--color-primary);}
